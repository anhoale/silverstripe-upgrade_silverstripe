UpgradeSilverstripe::run

    Signature

    public function run([string $pathLocation = ".", [String $logFileLocation = "", [String $to = "3.0", [Boolean $doBasicReplacement = false, [Boolean $markStickingPoints = false, array $ignoreFolderArray]]]]] )

    Parameters

    $pathLocation — string
    - enter dot for anything in current directory.


    $logFileLocation — string
    - where should the log file be saved. This file contains all the details about actual changes made.


    $to — string
    - if you set this to, for example 3.0 then the code will be upgraded from 2.4 to 3.0.


    $doBasicReplacement — object
    - If set to false to show proposed changes on screen. If set to true, basic replacements (i.e. straight forward replace A with B scenarios will be made)


    $markStickingPoints — object
    - If set to false nothing happens, if set to true any code that need changing manually will be marked in the code itself.


    $ignoreFolderArray — array
    - a list of folders that should not be searched (and replaced) - folders that are automatically ignore are: CMS, SAPPHIRE, FRAMEWORK (all in lowercase)
    outputs to screen and/or to file
